# Code of Conduct

## Our standard

参与者应尊重不同语言、地域、阶层、族群、性别与创作传统，针对作品和方法提出具体意见，不攻击创作者本人。

项目不接受骚扰、歧视、威胁、公开他人隐私、未经许可传播剧本，或以讽刺和羞辱代替技术讨论的行为。

## Scope

本准则适用于 Issues、Pull Requests、Discussions、代码审查以及代表本项目参与的其他公开空间。

## Enforcement

请通过仓库的私密安全报告渠道提交涉及隐私、骚扰或安全的问题。维护者可以删除不当内容、限制参与或永久禁止严重违规者。

所有处理将尽量保护报告者隐私，并根据事实与影响保持一致。
