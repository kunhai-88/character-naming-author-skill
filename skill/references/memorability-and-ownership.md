# 记忆点与剧情占有

## 记忆点不是稀奇

角色名的记忆力来自“可压缩的区别”，不是生僻字。观众应能用一句简单事实记住它：排行、乳名、家庭叫法、声音节奏、身份落差或某场关键叫名。

采用“九成真实底盘＋一成干净异常点”。异常点超过一个，名字会像设计品；没有异常点，名字容易真实但可替换。

## 六种可靠来源

### 1. 排行与家内位置

数字、长幼、小名和家中位置天然容易记，但必须有家庭事实支撑。排行不是复古装饰。

### 2. 公私双名

户口名可以普通，关系名负责记忆。家人叫乳名，同事叫姓或职称，对手喊全名。不同称呼揭示关系变化。

### 3. 生活偶然

登记同音字、长辈坚持、出生时的一件小事、兄姐留下的命名惯例，都能制造真实的不整齐。偶然应轻，不写成传奇身世。

### 4. 年代残留

名字保留父母那一代的政治、单位、教育或流行文化痕迹。不要把今天的审美倒灌给过去，也不要把年代感做成刻板词表。

### 5. 声音轮廓

常见字也能形成独特节奏。检查音节数量、开口度、轻重、重复和喊全名时的力度。避免全剧核心人物都是同长度、同尾音、同一类柔美音节。

### 6. 自然反差

父母的朴素愿望与人物后来命运可以形成反差，因为父母无法预知未来。不要为了反差反推一个过于精准的名字。

## 称呼系统

为主要人物建立称呼表：

| 关系 | 实际称呼 | 暴露的信息 |
|---|---|---|
| 长辈 | 本名、乳名或排行 | 家庭位置 |
| 亲密关系 | 私密简称 | 亲疏变化 |
| 同事邻里 | 姓、职称或外号 | 社会位置 |
| 对手 | 全名、只叫姓或拒绝称名 | 权力与敌意 |
| 陌生机构 | 法定全名 | 人物被社会登记的样子 |

称呼不能为了丰富而丰富。每个叫法都要由真实关系产生，并在剧本里实际使用。

## 剧情占有五步

1. **首次登记**：通过动作、关系或冲突出现姓名，不用人物互相自我介绍。
2. **稳定重复**：让一个最自然的称呼成为观众抓手。
3. **关系分化**：不同人物使用不同叫法，观众听称呼就知道关系。
4. **关键变调**：一直叫乳名的人第一次喊全名，或拒绝叫名的人终于叫出。
5. **结尾回收**：同一称呼因人物经历获得新含义，不必解释字义。

## 十分钟复述测试

模拟读者合上人物表十分钟后回答：

- 主角叫什么？
- 谁会怎样叫他？
- 为什么这个名字不是另一个普通名字？

如果只能记住“一个姓杨的普通男主”或依赖字义说明，复述关不通过。

## 两类常见失败

### 真实但无记忆

名字符合年代、能上户口、任何对白都顺，却可以无损搬进十部同类剧。补救方向不是加漂亮字，而是回到排行、称呼、家庭不整齐和声音轮廓。

### 有记忆但不像真人

名字醒目、寓意完整、视觉漂亮，却找不到剧中取名者。删除作者意图，先重建真实底盘。
